    Derivata is a collaboration in the same vein as Stromata, but between Lolita Fatjo and me.

    # Rules

    The rules for Derivata are the same as those for Stromata, basically no rules except that the next iteration within a strain must build upon the previous in some way.

    The first version, a-000.mp3 was sent by Lolita Fatjo on 2022-01-24 at 06:49 CET.
