Case

2020-10-10 46 Lua, Markdown spitlo/dotfiles/tree/master/.config/micro/plug/case

Case is a plugin for the micro editor which I’m trying to switch to from nano for quick edits. It converts text between different case styles, like kebab case or pascal case.

It’s pretty basic at the moment, and only supports single cursors.
